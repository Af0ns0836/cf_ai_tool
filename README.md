# cf_ai_tool
Github repository for AI-powered application on Cloudflare. Please refer to [PROMPTS.md](PROMPTS.md), where the utilized prompts to  build this project are described.

# Requirements

An AI-powered application should include the following components:
* LLM (recommend using Llama 3.3 on Workers AI), or an external LLM of your choice
* Workflow / coordination (recommend using Workflows, Workers or Durable Objects)
* User input via chat or voice (recommend using Pages or Realtime)
* Memory or state
